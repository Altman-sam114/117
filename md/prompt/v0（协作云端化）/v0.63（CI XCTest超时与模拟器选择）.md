# v0.63 / CI XCTest 超时与模拟器选择

## 版本号

`v0.63`

## 版本分配依据

`v0.62` 已 push（`0adc473`），但对应 GitHub Actions run `29165393920` 在 XCTest 阶段挂起，最终被取消，未能产出可验收 artifact。本轮不扩大 UI 范围，先修复云端验证链路，使结果包可在有限时间内完成，因此递增为 `v0.63`。

## 背景

v0.62 的 static checks、generic iOS build 和 Mac Catalyst build 均 success，但 XCTest 在模拟器 destination 上长时间 `in_progress`，无 artifact 上传，阻塞 Agent C 验收和后续 Mac/UI 迭代。

## 目标

- 为 CI job 增加 `timeout-minutes: 25`，避免无限挂起。
- XCTest 增加 `-destination-timeout 60` 与 720 秒 shell alarm，限制模拟器 boot/test 阻塞时间。
- 模拟器选择优先 `iPhone 16`，否则取较新 iOS runtime 的可用 iPhone。
- `VERSION` 更新为 `v0.63`。
- 同步 README、测试规范、核心流程和更新日志。

## 非目标

- 不修改业务 Swift 源码、数据模型、Markdown 语义或 UI 布局。
- 不运行本机 Xcode / xcodebuild / Simulator。
- 不回滚 v0.62 已 push 的 UI accessibility 改动。
